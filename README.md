# Fundamentos Asp.Net

Repositório criado com projeto de estudo do curso "Fundamentos do ASP.NET 6", do balta.io.

Assuntos abordados:
- Padrão MVC
- Health Check
- EF
- Arquitetura MVVM
- Autenticação e Autorização via Token (JWT Token gerado via JWTKey) e ApiKey
- Envio de e-mail
- Upload de arquivos estáticos
